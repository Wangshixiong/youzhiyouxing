# 体检发现异常，该怎么买保险？

买保险，很让人头疼的一件事情是健康告知。

就算没有住院过，也难免会有门诊或者是体检发现了些小异常，绕不过健康告知这个弯。

而健康告知就是，每个字我都认识，连在一起就不知道它在说啥了……

面对这种情况，最简单的方式是，交给别人，让别人来操心。

当然，这只是开玩笑。我来说说自己平时帮助身体有异常的客户投保，都是什么思路。一方面，方便大家做选择；另一方面，如果大家也可以了解下我是以何种思路帮你梳理相关问题的。

虽然一般一上来，大家都有自己的意向险种了，比如医疗险或者重疾险。

但我还是会习惯性问一下，其他险种有没有买？考虑一起投保吗？

虽然有点像推销，但……真的是为了你们好～

健康告知里除了有问到身体情况，通常还会有这么一条。

> 保险情况：被保险人过去 2 年内曾被保险公司拒保、延期、加费或者附加条件承保。

保险情况：被保险人过去 2 年内曾被保险公司拒保、延期、加费或者附加条件承保。

不同产品对身体要求不一样，健康告知的宽松度也不一样。

如果说先去投保了可能被拒保、延期、加费或者附加条件承保的产品，之后再投保其他的，就容易被健康告知问住。

而如果变换一下投保顺序，先买上健康告知宽松的，就不用担心投保其他险种受到这条询问的影响。

跟大家聊方案的时候，我最害怕听到的两个字是，对了……

「对了，我之前有体检过。」 「对了，我社保卡给我妈用过。」

这些话都让我「眼前一黑」，啊，之前的方案大概是白做了～

所以，我一般会在确认异常这一步，问得非常详细。

建议大家在买保险之前，可以自己先整理一遍住院和门诊的就医原因及相关病历、既往的体检报告有哪些异常，以及回忆一下是否有社保卡外借的情况。可以大大提高投保效率。

能够通过健康告知或者智能核保，就相当于可以直接买。

先说健康告知，不同产品健康告知会有差异性，可以先看看有没有符合健康告知的产品。

如果有，那就直接买。

再说智能核保，现在大部分产品都可以在线智能核保，选择对应的异常，完成问答就可以看符不符合投 保要求。

如果智能核保可以通过的话，也直接买就行了。

但也很容易在智能核保这里卡住，有两种情况：


* 因为缺少复查或者异常不符合投保要求被劝退
* 自己的异常在智能核保里找不到

因为缺少复查或者异常不符合投保要求被劝退

自己的异常在智能核保里找不到

如果异常十分轻微，可以尝试直接人工核保；如果异常可能会影响到核保结果，我会比较建议先进行针对性复查。

一定别全复查啊！

每次到了复查这一步，就会有人跟我说，最近我正好要体检，要不我体检之后再买吧？

别别别，通常情况下就复查需要的几个项目就好。万一又发现新的异常，得不偿失。

复查之后如果能通过智能核保了，就再走一遍智能核保就行。

如果不能的话，也就是说异常情况可能会影响到核保结果，九九会建议人工核保并且多家投保。

人工核保虽然有严也有松，但是整体来说它会比智能核保更灵活一些。

不同保险公司的核保会略有差异，比如有的对社保卡外借会宽松一些，有的对乙肝会宽松一些。多试几家，没准有一家就可以正常通过了。

好了，以上就是我针对有异常的客户如何投保的全部心得。

核保其实是非常个性化的事情，同一家公司不同时期、不同核保人员的的核保结果都会有所差异。对于我们来说，最好用的办法还是多比较多尝试。
