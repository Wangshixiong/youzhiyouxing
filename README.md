# 有知有行-E大理财知识爬虫项目

这是一个用于爬取有知有行网站上E大（ETF拯救世界）理财知识内容的Python爬虫项目。项目包含多个版本的爬虫脚本，可以获取投资理念、投资策略、人生哲学等多个板块的内容。

## 📋 项目简介

本项目旨在帮助用户批量获取有知有行网站上E大分享的理财知识内容，包括：
- **投资理念**：关于投资的基本理念和思考
- **投资策略**：具体的投资策略和方法
- **人生哲学**：关于金钱和生活的哲学思考
- **投资知识体系**：系统性的投资知识整理
- **投资第一课**：适合新手的基础投资课程

### youzhiyouxing-All3.0.py（推荐）

## 📦 安装依赖

```bash
pip install requests beautifulsoup4 lxml
```

## 🔧 使用方法

### 使用完整版爬虫（推荐）

```bash
python youzhiyouxing-All3.0.py
```

这个脚本会自动：
1. 爬取所有配置的内容合集
2. 创建完整的目录结构
3. 下载文章内容和图片
4. 生成索引文件

### 使用E大专版

```bash
python youzhiyouxing-image3.0.py
```

这个脚本专注于爬取E大干货合集的内容。

## ⚠️ 注意事项

1. **礼貌爬取**：脚本内置了1秒延迟，避免对目标网站造成压力
2. **版权问题**：爬取的内容仅供个人学习使用，请尊重原作者版权
3. **网络要求**：需要稳定的网络连接
4. **存储空间**：图片较多，建议预留足够存储空间
